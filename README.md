# Pogoda — додаток погоди (Weather App)

Vue 3 + Vite. Прогноз по Open-Meteo, геолокація, пошук міст (Україна / весь світ), почасовий прогноз, анімації погоди.

## Запуск

```bash
cd weather
npm install
npm run dev
```

Відкрити в браузері URL від Vite (наприклад http://localhost:5173).

## Збірка

```bash
cd weather
npm run build
```

Файли збираються в `weather/dist/`.
